1. First install my R3C0NH4CK3R tool

3. mkdir tools
4. cd tools
5. git clone https://github.com/karthi-the-hacker/R3C0NH4CK3R.git
6. cd R3C0NH4CK3R
7. chmod +x *.sh
8. ./install.sh
